FoodRescue Full-Stack Web Application - Development Prompt for Windsurf
Project Brief
Build a production-ready FoodRescue platform - an AI-powered web application connecting food donors (restaurants/stores) with NGOs and volunteers to redistribute surplus food. Create a modern, aesthetically stunning platform with smooth animations, professional UI/UX, and comprehensive functionality.

Tech Stack
Frontend
React.js 18+ (functional components, hooks)
React Router v6
Axios
Framer Motion (animations)
Tailwind CSS (styling)
Recharts (data visualization)
React-Leaflet (maps)
React Hot Toast (notifications)
Lucide React (icons)
React Hook Form + Zod (validation)
Context API + useReducer (state)
Socket.io-client (real-time)
Backend
Node.js + Express.js
MongoDB + Mongoose
JWT + Bcrypt
Multer + Cloudinary (images)
Socket.io (real-time)
Nodemailer (emails)
Node-cron (scheduled tasks)
AI Service
Python Flask microservice
scikit-learn (ML models)
pandas (data processing)
